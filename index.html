A steering wheel for racing games from a metal construction set and shelf planks. It usues an Arduino Uno board and an open source library. The biggest hurdle to overcome was the communication between the board and the computer, as the microproccessor doesn't natively support the USB protocol.

3-Axis Accelerometer 3D Visualization in Real Time

A GY-521 board connected to an Arduino Uno board and the communication is esbalished using I2C protocol. From here a a Node.js app, reads the data from the port, serves static files to the front end, establishes socket connection,handles the 3D Visualization and hosts the web page. This is the project I struggled the most with, as there were plenty of moving parts, most of which I had never dealt before.

A Tabletop Variable Voltage Suppy

I bought a broken ATX power suppy, fixed it , modded it so it suits my needs by having 3.3V, 5V, 12V and a variable voltage using a potentiometer, some LED's to indicate different states: Stand by and Power on, which can be changed using a switch. I use it to power most of my curcuits I work on and experiment with.
